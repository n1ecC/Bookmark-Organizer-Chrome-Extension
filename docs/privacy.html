Privacy Policy

Last Updated: June 9, 2026

This Privacy Policy describes how the AI Bookmark Organizer browser extension ("we", "us", or "our") handles your information.

1. Data Collection and Usage

We do not collect, store, or share any of your personal data.

The extension operates with a "Privacy First" philosophy:

  • Local Processing: All bookmark analysis and organization logic runs locally within your browser's memory.
  • No Remote Storage: We do not have servers, databases, or analytics trackers. Your bookmarks are never sent to us.

2. Third-Party Services (AI)

To provide intelligent categorization, the extension communicates directly with the OpenRouter API (which may route to Google Gemini or other models depending on your selection).

  • Direct Connection: Your API Key and bookmark titles are sent directly from your browser to OpenRouter. They do not pass through our servers.
  • Ephemeral Data: The data sent to the AI model is used solely for the purpose of categorization and is not retained by us.

Please review OpenRouter's Privacy Policy to understand how they handle API requests.

3. Permissions

The extension requires the following permissions to function. Each is used only for the purpose described below, and for no other purpose:

  • bookmarks: To read your existing bookmarks and write the newly organized folder structure back into the browser. This is the extension's core function.
  • storage: To locally store your OpenRouter API key and your theme/model preferences on your device so you don't have to re-enter them. This data never leaves your browser.
  • downloads: To export your organized bookmarks as an HTML file when you process an uploaded bookmarks file, so you can save the cleaned-up result.
  • sidePanel: To display the extension's interface in the browser's side panel.
  • Host permission (https://openrouter.ai/*): To send your bookmark titles and URLs directly from your browser to the OpenRouter API for AI categorization. The extension has no backend server of its own, so it must make these requests directly from the client. This permission is scoped to only openrouter.ai — the single endpoint that performs the categorization — and no data is sent to any other host.

4. Remote Code

This extension does not download or execute any remote code. It only transmits your bookmark data to the OpenRouter API and receives categorization results in return. All executable code is contained within the published extension package.

5. Contact

If you have questions about this policy, you may contact the developer via the GitHub Repository.
